// How It Works

Intelligent Compression Pipeline

HyperPress is not a new algorithm. It is an intelligence layer that profiles your data and races multiple preprocessing + backend combinations to find the optimal compression for each block.

Step 01
Input Data
Raw bytes of any type — text, binary, JSON, sensor data
Step 02
Block Analysis
Entropy, delta-entropy, null-ratio, structure detection per block
Step 03
Preprocessing
7 transforms race: Delta, BWT+MTF, RLE, Transpose, Nibble-Split
Step 04
Backend Racing
5 backends compete: zlib-9, LZMA, bz2, zlib+LZMA chain, stored
Step 05
Winner Output
Smallest result wins. CRC32 integrity. HPRX binary format.

7 Preprocessing Transforms

Transform 1

Delta Encoding

Stores byte-to-byte differences. Ideal for sensor signals.

Transform 2

Delta-16

16-bit delta for 2-byte aligned structured data.

Transform 3

BWT + MTF

Burrows-Wheeler + Move-to-Front for text data.

Transform 4

Run-Length Encoding

Collapses zero-runs. Perfect for sparse binaries.

Transform 5

Transpose

Column-wise reorder for structured records.

Transform 6

Nibble Split

Separates high/low nibbles to reduce entropy.

Transform 7

Identity

No preprocessing. Sometimes raw compresses best.

// Architecture

Block-by-Block Processing

Input Data
Raw bytes of any type
Text, binary, structured, or mixed data
Adaptive Block Splitter
Entropy-guided variable block sizing
4KB–128KB based on entropy variance
Block Profiler
Computes 6 features per block
ent, delta-ent, null-ratio, txt, struct, uniq
Candidate Selector
Context-aware preprocessing + backend pairing
3–15 candidates per block via rules engine
Racing Engine
Try all candidates, smallest output wins
Per-block CRC32 integrity verification
HPRX Output
Self-contained compressed stream
16B header + 14B per block header + data

// Quick Start

Three Lines of Code

from hyperpress import compress, decompress
 
compressed = compress(data)
restored = decompress(compressed)

Install: git clone https://github.com/BEKO2210/HyperPress.git && pip install -e HyperPress/ — Zero dependencies. Python 3.8+.
